# PHARMACIE EBEMA 💊

Application web de gestion de pharmacie — **PHARMACIE EBEMA**.

## 🚀 Fonctionnalités

- **Tableau de bord** : vue d'ensemble des ventes du jour, du stock et des alertes de rupture
- **Gestion des médicaments** : ajout, modification, suppression et recherche de produits
- **Gestion du stock** : suivi des quantités, alertes de stock minimum, entrées de stock
- **Ventes / Caisse** : enregistrement des ventes, calcul automatique du rendu-monnaie
- **Historique des ventes** : liste paginée avec filtre par date et produit
- **Fournisseurs** : répertoire des fournisseurs avec coordonnées
- **Sauvegarde locale** : toutes les données sont conservées dans le `localStorage` du navigateur

## 🖥️ Utilisation

Ouvrez simplement le fichier `index.html` dans un navigateur moderne (Chrome, Firefox, Edge…).  
Aucune installation ni serveur n'est requis.

## 🛠️ Technologies

- HTML5 / CSS3 (Flexbox & Grid)
- JavaScript ES6+ (vanilla, sans framework)
- Bootstrap 5.3 (CDN)
- Bootstrap Icons 1.11 (CDN)
- localStorage (persistance des données)
